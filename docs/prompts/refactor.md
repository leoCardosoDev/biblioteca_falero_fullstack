## Refactoring Request Prompt

> **Context**: I need to refactor the following code to improve quality/maintainability without changing behavior.
>
> **Target Code**: `[INSERT FILE PATH OR PASTE CODE HERE]`
>
> **Instructions**:
>
> 1. Read **`app/docs/ai/refactor-patterns.prompt.md`** to assume the Refactoring Engine persona.
> 2. Read **`.agent/standards/STANDARD_REFACTORING.md`** to load valid rules.
>
> **Execute**:
>
> 1. **Analyze**: Identify code smells (Long Method, Feature Envy, Conditionals, etc.).
> 2. **Plan**: Generate a **Transform Map** showing:
>    - Smell Detected
>    - Rule/Pattern to Apply
>    - Justification
> 3. **Refactor**: Rewrite the code applying the transformations.
>
> **Constraint**:
> - strictly preserve behavior.
> - verify 100% compliance with `STANDARD_BACKEND.md`.
